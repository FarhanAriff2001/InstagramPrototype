# InstagramPrototype

## Overview
Developed a simplified Instagram clone with a React frontend and a Flask (Python) REST API backend. Implemented core social media features including user authentication, post creation, and commenting. Designed and exposed backend APIs using Flask to handle data flow between the client and server, enabling clean separation of concerns and maintainable frontend-backend integration.

## Features
- User authentication with login and session management
- Create, view, and delete posts with image support
- Comment on posts with real-time feed updates
- RESTful API backend with clearly defined endpoints for all core features
- React-based frontend consuming Flask API for a responsive single-page experience
- Clean separation of concerns between frontend and backend layers

## Tech Stack
- **Language:** JavaScript (Frontend), Python (Backend)
- **Frameworks / Libraries:** React, Flask, Flask-RESTful
- **Tools:** SQLite / SQL database, REST API, npm, pip
